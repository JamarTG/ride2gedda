# Welcome to Ride2Gedda

## Project info

**App Idea:**  
Ride2Gedda is a **community-driven Jamaican bus commute app** that helps users plan and navigate JUTC bus journeys efficiently. It combines **official schedules**, **community-reported updates**, and **AI-assisted predictions** to provide:  
- Real-time bus arrival estimates  
- Notifications for delays, disruptions, or overcrowding  
- Interactive dashboard with routes, stops, and predictions  
- Community reporting of bus conditions  
- Predictive insights for smarter commuting

## How can I edit this code?

There are several ways to edit your application:

**Use your preferred IDE**

You can work locally by cloning this repository and pushing changes. Make sure you have **Node.js & npm** installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

Steps:

```sh
# Step 1: Clone the repository using the Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd Ride2Gedda

# Step 3: Install dependencies.
npm install

# Step 4: Start the development server with auto-reloading.
npm run dev
